# KFIE Frontend (React + Vite)

Cyberpunk-themed UI for Kenyan Freelance Intelligence Engine.

## Development

- Install: `npm install`
- Run: `npm run dev` (http://localhost:3000)
- Build: `npm run build`

## Pages

- Home - landing
- Login - phone OTP (mock)
- Job Board - list of jobs
- Job Detail - view and apply
- Profile - user profile

## Notes

- Uses React Router for navigation
- Axios for API calls (proxied to backend)
- Tailwind CSS for styling (add Tailwind to actual build)
- Dark theme with cyan accents
